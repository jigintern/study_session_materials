演習1: 基本のTween

課題1: gsap.to() で右に400px移動（2秒）

ヒント: gsap.to(ターゲット, { x: 値, duration: 値 })

課題2: gsap.from() で上から降ってくる（opacity も 0→1）

ヒント: gsap.from(ターゲット, { y: 値, opacity: 値, duration: 値 })

課題3: gsap.fromTo() で x:-200 から x:400 まで

ヒント: gsap.fromTo(ターゲット, { from }, { to })

課題4: gsap.set() で背景色を赤に → gsap.to() で回転360°

ヒント: gsap.set() → gsap.to() の順番で呼び出す
