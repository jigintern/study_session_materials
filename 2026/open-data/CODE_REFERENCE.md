# ハンズオン：正解コードの見方

## はじめに：`template/` からスタート

まず **[`template/`](./template/) フォルダをコピー**して土台を用意します（`template/README.md` 参照）。`index.html` と `styles.css` は完成版と同じで、**`app.js` は骨格・定数・補助関数・`main()` を含み、パート4〜8の該当ブロック（`[4-*]`〜`[8-*]`）はコメントアウトされています。**パート4ではスライドを見ながら `app.js` にコードを**入力**します（主にブロックのコメント解除）。  
**CSS は授業では書かず**、テンプレートの `styles.css` をそのまま使う想定です。

授業中に「ここまでのコードがほしい」「完成版はどうなっているか見たい」というとき向けの **参照用ドキュメント** です。

## 使い方（おすすめ）

1. **まず `template/app.js`** を開き、該当パートの `[4-*]` 〜 `[8-*]` ブロックを見る。
2. コメントを外した後の形を確認したいときは、**リポジトリ直下の `app.js`** を見る。
3. ブラウザを再読み込みして動作を確認する。

**注意**

- ネットワーク越しに CSV / API を取りに行くため、**Live Server 等で `http://localhost` から開く**前提です（`README.md` の「ローカルサーバー」参照）。
- `template/app.js` には **授業用の補助コード**（デモ表示・ログ・段階的有効化の分岐）があり、リポジトリ直下の `app.js` は **完成版**です。
- ただし、**各メソッドのロジック本体は揃うように管理**しています。

## どこに正解コードがあるか

| 見たいもの | 置き場所 | 含まれる主な内容 |
| :-- | :-- | :-- |
| パート 4〜8 の段階的なコード | [`template/app.js`](./template/app.js) | `[4-*]` 〜 `[8-*]` のブロックごとにコメントアウトされた授業用テンプレート |
| 授業の最終版（メイン） | このディレクトリ直下の `index.html` / `app.js` / `styles.css` | 第8章手前までを含む **いちばん新しい完成例** |
| Open-Meteo 単体の例 | [`open-meteo.html`](./open-meteo.html) | 降水量 API を単独で表示するサンプル |
| 地図の例 | [`map.html`](./map.html) | Leaflet を使った地図表示サンプル |

第1章・第2章は問いの整理が中心で、専用のコードファイル分岐はありません。

第8章（改善・Leaflet・自由課題）は **正解が一つに決まらない**ため、[`map.html`](./map.html) や [`open-meteo.html`](./open-meteo.html) をヒントにしつつ、リポジトリ直下のファイルをベースに手を入れてください。
