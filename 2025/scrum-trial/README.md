# 体験してみよう！サイコロで学ぶスクラム開発

> この資料は、2026年1月に行う香川高専詫間キャンパスでの勉強会イベントに向けて作成されたものです。
> 所要時間目安: 40分
> 筆者: [@haruyuki-16278](https://github.com/haruyuki-16278)

## 目的

この資料では、アジャイル型開発の代表的な手法であるスクラム開発を、サイコロを使った体験型のゲームを通じて学習します。

## 準備

この資料の内容を体験するために必要なものは以下の通りです。

* サイコロ
  * 手元の端末でランダムに1から6までの数字を生成できる場合は、サイコロがなくても代用可能です。

## 困ったときは

* 質問がある場合、章節項に割り振られた通し番号といっしょに質問してもらえると対応しやすいです。
* 資料に誤字脱字や理論的な欠陥にお気づきなら、[リポジトリ](https://github.com/jigintern/study_session_materials)にIssueを作成してContributerに知らせてください。確認して対応します。
