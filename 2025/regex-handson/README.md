# 正規表現を用いたパスワード強度チェッカーを作ってみよう

## 概要

Webサービスなどで、「パスワードは8文字以上で、英大文字・小文字・数字・記号を1文字以上含むこと」といったパスワードチェッカーを見かけますよね？
これを**正規表現**を使ってゼロから実装してみるハンズオンです！

パスワードチェッカーを段階的に強化していくことで、正規表現の基礎から実践的な使い方まで学べます。

## 学ぶこと

- メタ文字・文字クラス・量指定子
- 先読み・後読み
- キャプチャグループ・後方参照

### レベルごとの学習ポイント

| レベル  | 内容                             | 正規表現の学習ポイント                                                          |
| ------- | -------------------------------- | ------------------------------------------------------------------------------- |
| レベル1 | 8文字以上                        | 量指定子 `{n,}`                                                                 |
| レベル2 | 英小文字を含む                   | 肯定先読み `(?=...)`, 文字クラス `[a-z]`                                        |
| レベル3 | 連続する同じ文字が3つ以上ない    | キャプチャグループ `(.)`, 後方参照 `\1`, 否定先読み `(?!...)`, アンカー `^` `$` |
| レベル4 | 英大文字と数字を両方含む         | 文字クラスのショートハンド `\d`, 複数の先読み                                   |
| レベル5 | 数字の直後に記号がある箇所を含む | 肯定後読み `(?<=...)`                                                           |

## 準備

### 必要なもの

- テキストエディタ（VSCodeなど）
- ブラウザ（Chromeなど）
- Deno 2.x

## ハンズオンの進め方

### 環境確認

1. `deno run --allow-read --allow-net --watch server.deno.js`
   を実行してローカルサーバーを起動
2. ブラウザで `http://localhost:8080/index.html` にアクセス
3. パスワード入力欄に文字を入力
4. すべて失敗していれば準備完了

### 関数を実装

1. `project/password-checker.js` を開く
2. 取り組むレベルの関数を実装

```javascript
export function checkLevel1(password) {
  return /.{8,}/.test(password);
}
```

### 関数をテスト

1. `deno test project/password-checker.test.js -- --level=N`
   （Nはレベル番号）を実行
2. すべてのテストが成功するように実装を修正

### 動作確認

1. `index.html` で色々なパスワードを試す
2. `password-checker.test.js` に新しいテストケースを追加してみる
